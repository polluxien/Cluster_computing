# Running the program

Der bereitgestellte Makefile lässt die Ausführung des Programms ganz einfach über 

make run-n laufen, wobei n die Anzahl der Prozessoren darstellt. 

Falls der build Ordner nicht automatisch erstellt wird empfielt sich ihn manuell auf der Ebene des Makefiles zu erstellen.

make clean löscht alle bisher erstellten Datein.

Die Bin datei liegt am Ende im build Ordner unter dem Namen: mpi_matrix_multiply.out
